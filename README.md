Student Result Management System (SRMS)
A simple and efficient C-based Student Result Management System that allows admins, students, and guests to securely access and manage student academic records.
The project demonstrates file handling, authentication, modular programming, and menu-driven interfaces in C.Features
Login System
1.Separate credentials for Admin, Student, and Guest
2.Secure verification using credentials.txt
3.Role-based access control
Student Module
1.Add new student records
2.View all student records
3.Search student by roll number
4.Update marks
5.Delete a record
Guest Module
1.View student results (read-only)
2.No modification access
 File Management
1.Uses three separate files:
2.students.txt → Stores student data
3.guest.txt → Stores guest data (if needed)
4.credentials.txt → Stores login usernames & passwords
Technologies Used
C Programming
1.File Handling (txt-based database)
2.String Handling
3.Modular Menu-Driven Architecture
